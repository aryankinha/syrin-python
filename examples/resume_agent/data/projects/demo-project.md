# Demo Project: Voice AI Agent

Built a voice AI agent using Syrin and Pipecat. The agent answers recruiter calls, searches a knowledge base for resume and project info, and can schedule meetings.

- **Tech**: Python, Syrin, Pipecat, Deepgram, ElevenLabs, Twilio
- **Outcome**: Conversational phone agent with RAG-backed answers
- **Link**: examples/resume_agent in syrin-python repo
